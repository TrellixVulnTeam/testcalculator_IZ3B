program kalkulator ini menerima input berupa angka dan dimasukkan ke dalam array untuk menghasilkan beberapa output

jenis output:
- array         = mengeluarkan semua angka yang telah diinput
- sorted array  = mengeluarkan semua angka yang telah diinput dengan urutan dari kecil ke besar
- multiplied    = mengeluarkan total perkalian dari semua angka yang terdapat di array
- average       = mengeluarkan hasil rata-rata dari semua angka yang terdapat di array
- middle number = mengeluarkan nilai tengah dari array
- total         = mengeluarkan total penjumlahan dari semua angka yang terdapat di array

cara penggunaan:
- angka dapat diinput menggunakan keyboard atau mengklik nutton yang tersedia
- klik tombol "+" untuk menambah angka yang telah diketik ke dalam array
- klik tombol "-" untuk menghapus angka terakhir di dalam array
- klik tombol "C" untuk menghapus semua isi array
- klik tombol "=" untuk mengeluarkan semua hasil perhitungan

limitasi program:
- menginput karakter selain angka akan menyebabkan multiplied, average, dan total untuk menghasilkan error karena program tidak dapat menghitung karakter selain angka
- middle number akan mengeluarkan data yang berada di tengah array walaupun data tersebut berupa string
- program ini tidak akan bekerja apabila browser user tidak menjalankan javascript
